# ルビの出版社公開ルール調査（2026-07-16）

## 結論

- 週刊少年ジャンプについて公開資料で確認できた明確な運用は「掲載漫画は総ルビ」
  「通常の読みは編集部・印刷所が指定」「当て字や特殊な読みだけ作者が指定」の3点。
- ユーザー判断により、B-MANGAへ総ルビ機能は追加しない。
- ルビのサイズ、親文字との間隔、字間、行送り、中央／肩付き、モノ／グループ／
  熟語の使い分けについて、ジャンプ固有の公開数値は確認できなかった。
- したがって「ジャンプ独自値」を推測して初期値にはしない。公開されている一般的な
  日本語組版原則を根拠に、既存の初期値を維持する。

## 確認した一次資料

### 集英社・週刊少年ジャンプ

- [JUMP新世界漫画賞 Q&A](https://www.jump-mangasho.com/qa/)
  - ジャンプ漫画は総ルビで、通常の読みは編集部・印刷所側が指定し、作者は当て字や
    特殊な読みだけ指定する旨を公開している。
- [共同印刷「週刊少年ジャンプができるまで」](https://www.kyodoprinting.co.jp/jump/receiving.html)
  - 完成原稿に編集者が文字サイズや書体等を指定して印刷工程へ渡す流れを公開している。
  - 公開ページには、ルビ固有のサイズ・間隔等の数値はない。
- [手塚・赤塚賞 応募要項](https://www.shonenjump.com/j/manga-shou/teaka.html)
  - 原稿寸法やデジタル原稿の解像度等は明記されているが、ルビ組版の数値指定はない。

### 他出版社の公開応募要項

- [小学館 新人コミック大賞](https://shincomi.shogakukan.co.jp/)
- [講談社 週刊少年マガジン 新人漫画大賞](https://debut.shonenmagazine.com/awards/)
- [KADOKAWA 角川漫画新人大賞](https://promo.kadokawa.co.jp/kadokawamanga/)

いずれも原稿寸法、解像度、形式等の応募条件は公開しているが、漫画のルビサイズ・
間隔・配置に関する数値規則は確認できなかった。出版社・媒体の内部組版基準が存在
しないという意味ではなく、一般向け一次資料からは特定できない、という調査結果である。

参考として、講談社の児童文学新人賞は、読みにくい名前や特殊な読み以外は応募原稿に
ルビ不要と明記しており、対象読者・媒体によって運用が異なることも確認できる。
[講談社 児童文学新人賞 Q&A](https://cocreco.kodansha.co.jp/cocreco/general/books/award/DItOB)

## 一般的な日本語組版原則とB-MANGAの初期値

[W3C「日本語組版処理の要件」](https://www.w3.org/TR/jlreq/)は主としてJIS X 4051に
基づき、[W3C「日本語ルビの簡便な配置ルール」](https://www.w3.org/TR/simple-ruby/)
は次の既定を示している。

| 項目 | 公開された一般原則 | B-MANGA初期値／対応 |
|---|---|---|
| サイズ | 原則、親文字の1/2 | 50%（維持） |
| 付ける側 | 横書きは上、縦書きは右 | 実装済み（維持） |
| 親文字との間隔 | 親文字の文字枠へ密着 | 0em（維持） |
| モノルビ | 親文字と読みの中心を揃える | 中付き（維持） |
| 短いグループルビ | 前後1：ルビ文字間2。前後は親文字の二分を上限 | 今回不足を修正 |
| 長いルビ | 前後の文字種等を考慮してはみ出しを調整 | 既存の衝突回避を維持 |

## 実装方針

- 初期値は50%、0em、ルビ字間0、中付き、本文フォント継承を維持する。
- 短い日本語グループルビだけ、前後1：文字間2で親文字幅へ配分する。
- 前後の空きが親文字の二分を超える場合は二分で止め、残りを文字間へ回す。
- 欧文の読みは字間を広げず、語のまとまりを保って親文字の中央へ置く。
- 肩付き、モノルビ、親文字より長いルビ、手動ルビ、当て字、登録辞書は変更しない。
- ジャンプ固有の非公開値を推測しない。編集部から個別指定を受けた場合は、既存の
  サイズ・間隔・字間・行送り・配置設定で作品／プリセットごとに合わせる。
