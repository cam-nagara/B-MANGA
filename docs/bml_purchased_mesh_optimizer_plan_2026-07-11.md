# B-MANGA Liner 購入素材メッシュ最適化 計画

作成日: 2026-07-11
対象: Blender 5.1.2 / B-MANGA Liner
状態: 完了（B-MANGA Liner v0.3.196）

## 目的

三角面主体の購入素材を、テクスチャ付きの最終レンダリング用メッシュとして
滑らかにし、その同じ実メッシュからアウトライン・稜谷線・交差線を生成する。
ライン反映と「中間頂点用ライン細分化」からは独立した明示操作にする。

## 確定要件

- 元メッシュの永続的な退避コピーを blend 内へ残さない。
- 処理中だけ候補メッシュを作り、全対象の検証合格後に一括確定する。
- 一件でも不合格なら候補を破棄し、選択対象を一切変更しない。
- 三角面・四角面・Nゴン・開口境界を受け付ける。
- UV、素材順、面の素材番号、カスタム法線、鋭角を維持する。
- ライン用だけの代理メッシュは作らない。表示・レンダリング・ライン解析は
  確定後の同一メッシュを参照する。
- リンク素材、シェイプキー、変形用頂点グループ、判定不能な非多様体や
  属性の異なる重複面は、安全な自動処理対象外として一括確定を止める。

## 処理方式

1. 選択したローカル静的メッシュを診断する。
2. 同じ向き・素材・UVを持つ完全重複面とゼロ面積面だけを候補上で除去する。
3. Blender の固定三角化結果から、三角面・四角面・Nゴンを三角パッチへ統一する。
4. 各頂点位置とコーナー法線から三次曲面を推定し、曲率がある辺だけを局所分割する。
5. UVは元三角面内の線形補間、素材番号は元面から継承する。法線不連続、
   素材境界、UV境界、非多様体辺は曲面をまたいで平均しない。
6. 面数上限、有限座標、素材範囲、UV層、法線、境界、候補の妥当性を検査する。
7. 全対象が合格した場合だけメッシュを一括差し替え、旧メッシュに利用者が
   いなければ削除する。既存ラインは作成待ちにする。

## 画面

BMLiner タブへ独立した「購入素材メッシュ最適化」パネルを追加する。

- 品質: 標準 / 近接用
- 「選択メッシュを最適化」
- 前回結果: 最適化数、変更不要数、削除面数、追加頂点数、不合格理由

購入素材の修正前に実行する工程であることを明記し、ライン反映を暗黙実行しない。

## 検証ゲート

- 合成素材: 三角、四角、混在、開口、完全重複、逆向き重複、ゼロ面積、
  3面共有非多様体、複数素材、UVシーム、カスタム法線。
- 原子性: 不合格を混ぜた選択では全対象が不変。
- 保存再読込: 頂点・面・UV・素材・法線・設定状態が一致。
- 全線種: 最適化後にアウトライン・稜谷線・交差線・選択線・バンプ線を
  個別および同時反映し、例外・黒ベタ・飛び出し・素材面隠れがない。
- 指定実素材: Japanese_Streetscape_Tokyo_0004.blend を保存せず読み、
  代表対象と全体診断で性能・形状・テクスチャを確認する。

## 実装結果

- 「購入素材メッシュ最適化」をライン反映とは独立した明示操作として追加した。
- 元頂点を固定し、コーナー法線から曲率が確認できる辺だけを局所分割する。
- 素材・UV・分割法線・色・シャープ辺・UVシーム・クリース・生成座標の
  テクスチャ空間を候補へ継承する。素材・UV・法線の境界はまたいで平均しない。
- 候補は全選択対象の検証後に一括確定する。不合格時は候補をすべて破棄する。
- 最適化済みメッシュでは「中間頂点用ライン細分化」を重ねず、確定した表示
  メッシュをアウトライン・稜谷線・交差線・選択線・バンプ線の共通入力にする。
- 表裏重複、属性差のある重複、3面以上が共有する辺、孤立辺、未対応属性、
  リンク素材、モディファイア、シェイプキー、頂点グループは黙って変更せず中止する。

## 検証結果

- Blender 5.1.2で、三角面・四角面・Nゴン・開口境界・完全重複・ゼロ面積・
  逆向き重複・非多様体・孤立辺・複数素材・UV・色・カスタム法線・保存再読込を確認した。
- 全線種の同時表示、Eevee / Cyclesのバンプ線、旧ライン細分化、全反映、
  素材スロット順、登録解除・再登録の回帰テストが合格した。
- 指定実素材162メッシュ（元101,124面）を保存せず診断し、不合格0、最適化123、
  変更不要39、候補生成約21秒だった。代表マンホールは49頂点・48面から
  169頂点・288面となり、テクスチャ維持と円周の滑らかさを画像で確認した。
- AI目視で黒ベタ、飛び出し、素材面隠れ、全線種の異常がないことを確認した。
