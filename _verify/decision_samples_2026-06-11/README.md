# 判断が必要な項目の比較サンプル (2026-06-11)

全機能チェックで見つかった「機能していない/整理が必要なUI項目」のうち、仕様判断が必要な4件の比較画像です。
それぞれ案を選んでいただければ、その方針で実装します。

## 判断1: 「角を尖らせる」 (`判断1_角を尖らせる_形状別の効き.png`)

フキダシ詳細設定のチェックボックス。現状はトゲ(直線)・トゲ(曲線)にだけ効き、雲ともやもやではオン/オフが同じ見た目です
(雲の谷の尖りは参考画像合わせで確定済みのため、常に尖る仕様)。

- **案a**: 雲・もやもやのときはチェックボックスを非表示にする
- **案b**: 表示は残し、説明文に「雲・もやもやの主線形状には影響しない」と明記する

## 判断2: フキダシ白抜き線の「束の幅 (mm)」 (`判断2_白抜き線_束の幅.png`)

上段=現状のフキダシ白抜き線。束の幅 3/10/30mm のどれを入れても変化しません
(線の配置が「本数×間隔」で決まるため)。下段=幅が効く場合のイメージ
(同じ生成方式の効果線・白抜き線で、本数自動計算にして幅だけ変えたもの)。

- **案a**: フキダシ側から「束の幅」項目を削除し、本数と間隔だけにする
- **案b**: 幅を正として効かせる (本数を手動指定したときは間隔を幅から自動算出する)

## 判断3: 効果線の入り抜き範囲 (`判断3_効果線_入り抜き範囲.png`)

「範囲モード」「入りの範囲(%/mm)」「抜きの範囲(%/mm)」は現状無効で、「入り始点/抜き始点」だけが効きます。
さらに「入り(%)」は入り始点が既定の0%のままだと無反応です。

- **案a**: 「範囲モード」「入りの範囲」「抜きの範囲」を削除し、「入り始点/抜き始点」に一本化する
  (あわせて入り始点の既定を例えば50%にして「入り%を動かせばすぐ効く」状態にする)
- **案b**: 長さ(mm)指定を実装し直す (ウニフラ・白抜き線の確定済みの見た目に影響しない範囲で)

## 判断4: 見開きページのページ一覧表示 (`判断4_見開き_ページ一覧の幅.png`)

見開きのプレビュー内容は今回の修正で正しい縦横比になりましたが、タイル自体は1ページ分の幅のままなので
小さく表示されます。右の「案」は2スロット分の幅で表示した場合のイメージ(合成モックアップ)です。

- **案a**: 現状のまま (1ページ幅のタイルに縮小表示)
- **案b**: 見開きをスロット2つ分の幅で表示する (ページ配置グリッド全体の改修が必要・工数大)

## 参考: 判断不要で削除提案のみの項目 (画像なし)

- フキダシ形状の未使用保存パラメータ5つ (雲の波数/波の振幅/トゲ数/トゲの深さ/トゲ乱れ) → 削除提案
- 効果線白抜き線の「長さ乱れ/最小長さ(%)」 → 無効のため削除提案
- 作品パネルのコマ用blendファイル解説文 → ツールチップ化提案
